# User Stories

## Opportunity Intake

- As a user, I can paste a job-offer link into chat so the agent creates an opportunity record.
- As a user, I can paste several job-offer links at once so the agent queues and imports them as separate opportunities.
- As a user, I can paste a raw job description so the agent creates an opportunity without requiring a URL.
- As a user, I can ask the agent to enrich a newly added opportunity with title, company, location, description, seniority, compensation, and source metadata.

## Opportunity Table

- As a user, I can view all opportunities in a table so I can understand my pipeline at a glance.
- As a user, I can sort and filter opportunities by score, company, title, status, applied state, source, and updated time.
- As a user, I can ask the agent questions about the table, such as "show me unapplied roles above 80" or "which ones should I apply to today?"
- As a user, I can select an opportunity to see details, extracted facts, notes, scoring rationale, and application materials.

## Agent-Managed Workflow

- As a user, I can ask the agent to score each opportunity against my profile or resume.
- As a user, I can ask the agent to explain why a role is a strong or weak fit.
- As a user, I can ask the agent to update opportunity status, such as interested, rejected, applied, interviewing, offer, or archived.
- As a user, I can require approval before the agent performs important mutations, such as marking an opportunity applied or replacing generated application materials.

## Application Materials

- As a user, I can ask the agent to generate cover-letter bullets for a selected opportunity.
- As a user, I can ask the agent to draft a full cover letter using the opportunity and my profile.
- As a user, I can save generated material back to the opportunity so I can revisit it later.
- As a user, I can ask the agent to prepare interview notes or company research for an opportunity.

## Profile And Preferences

- As a user, I can paste my resume into chat so the agent can ground scoring and generated materials in my background.
- As a user, I can have the system extract sections, skills, experience, education, projects, and links from my pasted resume text.
- As a user, I can later upload a PDF resume once file ingestion is supported.
- As a user, I can provide job-search preferences, such as target roles, location, remote preference, compensation, industries, and deal breakers.
- As a user, I can review and edit the profile brief the agent uses for matching opportunities.
- As a user, I can ask the agent to update my preferences as my search changes.

## UI Direction

- As a user, I can use chat as the primary command surface while the opportunities table stays visible.
- As a user, I can see agent-created or agent-updated records appear in the table without manually copying data.
- As a user, I can open generated artifacts, such as cover letters or research notes, beside the chat.
